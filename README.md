## Capstone project of udacity frontend nano
### Travel Assistant

This app helps you plan your trip!


## APIs Used (register your own api key in order to use this app)
1. Pixabay
2. GeoNames
3. WeatherBit

## Tools involved
- Javascript
- Node.js
- Express
- Webpack

### Install

start dev server
```shell script
npm install
npm run build-dev
npm start
```
or start prod server
```shell script
npm run build-prod
```
